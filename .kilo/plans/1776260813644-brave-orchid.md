# Plan: Add PDF and Excel Downloads with Map Image

## Goal
Implement download functionality for PDF and Excel formats, ensuring the map image is included in the downloads. The existing Excel download should be enhanced if needed, and a new PDF download option added to the dropdown menu.

## Current State Analysis
- Excel download already exists via `dcc.Download`, generating `.xlsx` files with filtered data (all, high, medium, low priority).
- Map is generated using `plotly.express.choropleth_mapbox`.
- No PDF download currently implemented.
- Map image is not included in any downloads.

## Implementation Steps

### 1. Enhance Map Image Generation
- Modify the map generation callback to also export the plotly figure as a PNG image.
- Use `fig.write_image()` to save the map as a temporary file (e.g., in a temp directory).
- Store the image path in a dcc.Store for access during downloads.

### 2. Update Excel Download
- Excel downloads are data-only; no map image can be embedded in Excel easily.
- If map inclusion is required, consider adding a note or separate image download, but since user specified "gambar peta nya" (map image), focus on PDF for map inclusion.
- Keep existing Excel functionality as is.

### 3. Implement PDF Download
- Add PDF option to the dropdown menu: `dbc.DropdownMenuItem("Unduh PDF", id="dl-pdf")`.
- Use a Python PDF library like `reportlab` or `fpdf` to generate PDF.
- PDF structure:
  - Title: "Hasil Prioritas Kemiskinan Kabupaten/Kota Sumatera Utara"
  - Summary text (same as displayed in app).
  - KPI counts.
  - Data table (formatted nicely).
  - Map image (embedded PNG).
- Install required dependencies: `pip install reportlab` (or fpdf).
- Create a callback for `dl-pdf` that:
  - Retrieves current data and map image.
  - Generates PDF with reportlab.
  - Triggers download via dcc.Download.

### 4. Update Dropdown Menu
- Add the PDF option to the existing dropdown.
- Ensure IDs are unique (e.g., `id="dl-pdf"`).

### 5. Testing and Validation
- Test PDF generation with sample data.
- Ensure map image is correctly embedded and scaled.
- Verify downloads work for different filters (all, priorities).
- Check on different browsers/devices.

### 6. Dependencies
- Install `reportlab` and `kaleido` in the virtual environment: `pip install reportlab kaleido`.
- If using virtual environment (.venv), activate it first: `.\.venv\Scripts\activate` on Windows.
- Add `reportlab` and `kaleido` to requirements.txt for future installations.
